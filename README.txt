
# IusCivile Pro+ — Vercel (All‑in‑One)

Pubblica su **Vercel** con questi parametri:

1) Root Directory: `web/`
2) Build Command: `npm run build`
3) Output Directory: `dist`

Variabili (Project → Settings → Environment Variables):
- `OPENAI_API_KEY` = la tua chiave OpenAI (`sk-...` o `sk-proj-...`)
- `GOOGLE_API_KEY` = (opzionale) la tua `AIza...` per la ricerca Google
- `GOOGLE_CX`     = (opzionale) il tuo ID Programmable Search Engine
- `IUS_CHAT_MODEL`= (opzionale) default `gpt-4.1-mini`

Dopo il deploy, apri l'URL e usa **Condividi → Aggiungi a Home** su iPhone.
